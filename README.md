

## **Breast Cancer Analysis**

### Description:
This project classifies breast cancer cases into **Benign** and **Malignant** categories using **Logistic Regression**. The model achieved an impressive **92%** accuracy on the dataset, which is a strong indicator of its performance in detecting cancerous and non-cancerous cases. This analysis helps in early detection and diagnosis, crucial for medical purposes.

### Key Features:
- Classifies breast cancer cases into benign and malignant categories
- Utilizes logistic regression for classification
- Achieved an accuracy of 92% on the test data

### Technologies Used:
- Python
- Logistic Regression (scikit-learn)
- Pandas and NumPy for data manipulation
- Matplotlib and Seaborn for data visualization
